# Techfest 3D Interactive Website

An immersive and futuristic 3D interactive website created for the Techfest "Build in 3D" challenge.

## 🚀 Features
- Interactive 3D experience using Three.js
- Smooth animations powered by GSAP
- Responsive design for mobile, tablet, and desktop
- Modern futuristic UI/UX
- Dynamic transitions and scroll effects
- Optimized performance

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Three.js
- GSAP

## 📂 Project Structure
```
techfest-3d-interactive-website/
│── index.html
│── style.css
│── script.js
│── assets/
└── README.md
```

## 📌 Purpose
This project was developed as a submission for the **Techfest – Build in 3D** challenge, showcasing modern web development, 3D graphics, and interactive user experience.
